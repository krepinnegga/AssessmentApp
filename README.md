<h2>Running on Virtual Device</h2>
<ul>

   <li>Run "npx expo start" in terminal.</li>
   <li>Pressing "a" will open in an Android Emulator or connected device.</li>
  <li>Pressing "i" will open in an iOS Simulator.</li>
